- Gemfile
  > https://jekyllrb.com/docs/ruby-101/#gems \
  > Ver [Ruby GEMS](./RUBY.md)
  - Every Jekyll site has a **Gemfile** in the main folder.
  - **Example**:
    ```ruby
    source "https://rubygems.org"

    gem "jekyll"

    group :jekyll_plugins do
    gem "jekyll-feed"
    gem "jekyll-seo-tag"
    end
    ```
